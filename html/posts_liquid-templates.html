Understanding Liquid Templates

What is Liquid?

Liquid is a powerful templating language that allows you to create dynamic web pages. In Raytha, we use the Fluid library to render Liquid templates with full .NET integration.

Basic Syntax

Liquid uses two types of delimiters:

  • Double curly braces - Output tags for displaying content
  • Curly brace percent - Logic tags for control flow

Variables

Access your content data through variables like Target, ContentType, and CurrentOrganization.

Filters

Transform output with filters like the date filter to format timestamps.
