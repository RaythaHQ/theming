Getting Started with Raytha CMS

Post January 15, 2024 Updated January 15, 2024 Published Edit in Admin

Introduction

Welcome to Raytha CMS! This is a comprehensive guide to help you get started with building your first website using Raytha. In this article, we'll cover the basics of content types, templates, and how to publish your first content item.

What is Raytha?

Raytha is a powerful, open-source content management system built with .NET that gives you complete control over your content and templates. Unlike traditional CMS platforms, Raytha lets you define exactly how your content is structured and displayed.

Getting Started

To get started with Raytha, you'll need to:

  • Install .NET 8 or later
  • Clone the Raytha repository
  • Run the database migrations
  • Start the application

Creating Your First Content Type

Content types are the foundation of Raytha. They define the structure of your content with custom fields like text, images, relationships, and more. Head to the Admin Panel to create your first content type!
