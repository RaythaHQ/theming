Building Custom Content Types

Post February 1, 2024 Updated February 2, 2024 Published Edit in Admin

The Power of Custom Content Types

One of Raytha's most powerful features is the ability to create custom content types with your own fields. Whether you're building a blog, an e-commerce site, or a documentation portal, you can define exactly the fields you need.

Available Field Types

  • Single Line Text - Short text input
  • Long Text - Multi-line text
  • WYSIWYG - Rich text editor
  • Dropdown - Single select
  • Checkbox - Boolean toggle
  • Date - Date picker
  • Attachment - File upload
  • One-to-One Relationship - Link to another content item

Best Practices

Plan your content structure before creating types. Consider relationships between content types and how they'll be displayed in templates.
